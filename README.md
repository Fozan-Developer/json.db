## 📦 Install
---
**NPM**
```sh
npm instal @mr_fozan/json.db
```
**YARN**

```sh
$ yarn add @mr_fozan/json.db
```

## 🚀 Usage
---
```js
const db = require('@mr_fozan/json.db');

const connect = db.connect({ saveTime: 1000, models: [ "users" ] });
```
